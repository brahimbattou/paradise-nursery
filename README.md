# Paradise Nursery Shopping Application

## Project Name: Paradise Nursery

Paradise Nursery is a React and Redux-based e-commerce web application for an online plant shop. Users can browse a variety of houseplants, add them to a shopping cart, and manage their cart items.

## Features

- Browse houseplants organized by category
- Add plants to cart with dynamic cart count updates
- View cart with item quantities, individual costs, and total price
- Increase or decrease item quantities
- Remove items from cart
- Responsive navigation bar with links to Home, Plants, and Cart
- "Get Started" landing page

## Technologies Used

- React
- Redux Toolkit
- React Redux
- CSS3
